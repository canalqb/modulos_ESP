Configurações de Acesso à Nuvem (Google Cloud)

Para que o Opus salve seus dados com segurança, precisamos de acesso ao seu Google Drive pessoal. Siga o tutorial no Canalqb.com.br se tiver dúvidas.

1
Credenciais do Projeto

No Google Cloud Console, crie um credenciamento do tipo "App para Computador". Configure a URI de redirecionamento para http://localhost:3000/auth/callback.

2
Autorizar o Hub Opus

Adicione o seu e-mail pessoal como "Usuário de Teste" na tela de consentimento OAuth do seu projeto Google.

✅ Token Refresh Gerado e Salvo!
3
Pasta de Destino

Informe o ID da pasta no seu Drive onde o sistema irá morar.
